## 더 큰 숫자 반환하기

이 함수를 완성하여 더 큰 숫자를 반환하세요!

다음은 사용하지 마세요:

* `return`
* 다른 함수 호출
* 추가 변수

<div class="hint">
  원한다면 한 줄로 해결하는 것도 가능합니다!

  다른 언어의 유사한 예:
  - C(++)에서는: `a > b ? a : b`
  - Python에서는: `a if a > b else b`
</div>
<div class="hint">
  Rust에서 다음 사항을 기억하세요:

  * <code>if</code> 조건은 괄호로 둘러쌀 필요가 없습니다.
  * <code>if</code> / <code>else</code> 조건문은 표현식입니다.
  * 각 조건문 뒤에는 <code>{}</code> 블록이 따라옵니다.
</div>
